Explicación

El juego de Gato o tres en línea es un juego que se realiza entre dos jugadores, donde cada jugador tiene asignado un símbolo diferente (generalmente ❌ y ⭕). Se marcan espacios sobre un tablero de 3×3 cuadritos, tomando turnos alternados. El juego termina cuando alguno de los jugadores coloca tres de sus símbolos alineados, o ya no hay espacios vacíos en el tablero.

Estructura del Código

Clases Principales:

  • Clase Jugador:
    • La clase Jugador representa a un jugador en el juego.
    • Métodos:
      • __init__(simbolo: str): Inicializa un jugador con un símbolo específico ('O' o 'X').
  • Clase Tablero:
    • La clase Tablero representa el tablero del juego, que es una cuadrícula de 3x3.
    • Atributos:
      • tablero (list): Representa el estado del tablero.
    • Métodos:
      • void __init__(self): Genera un tablero de 3x3 vacío.
      • verificar(self, fila: int, columna: int): Verifica si una posición en el tablero está vacía. return Tablero
      • realizar_movimiento(self, fila: int, columna: int, jugador: str): Realiza un movimiento si la posición está vacía. return "True" o "False"
      • ganador(self): Define las combinaciones ganadoras. return "none"
      • empate(self): Verifica si hay un empate (todas las posiciones están ocupadas). return que todos los campos este vacios
  • Clase Juego:
    • La clase Juego inicializa la ventana del juego, controla el turno actual y genera los botones.
    • Atributos:
      • tablero (list): El tablero de juego.
      • jugador1 (str): Representa al primer jugador.
      • jugador2 (str): Representa al segundo jugador.
      • turno (int): Controla el turno actual del juego.
      • botones (list): Almacena los botones de la interfaz gráfica para cada celda del tablero.
    • Métodos:
      • void __init__(self, master: tk): Inicializa la ventana del juego.
      • void crear_botones(self): Crea los botones para cada celda del tablero.
      • void jugar(self, fila: int, columna: int): Realiza un movimiento y verifica si hay un ganador o empate, imprimiendo el mensaje correspondiente.
      • void reiniciar_juego(self): Reinicia el tablero y el turno.
